# WaterMark 水印方法

## 调用方法

```ts
  let waterMark = new Utils.WaterMark(['水印文字哈哈哈哈', '2022-12-08'], ['img', 'img2'])
  waterMark.startWaterMark();
```

## 参数

| 属性   | 说明 |   类型  | 默认值  | 版本  |
| :-------------: | :----------: | :------------: | :------------: | :------------: |
|第一个参数 | 水印内容     | Arrar  | ['水印文字哈哈哈哈', '2022-12-08'] | 1.2.2|
| 第二个参数       |    要绘制水印的 id    |        Arrar - string  | [] |1.2.2 |
